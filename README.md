# 雜魚小貓娘 (Zayu-NekoNeko-Bot)

「雜魚小貓娘」是一款集互動遊戲、社交娛樂與伺服器守護於一身的多功能 Discord 機器人。

## 功能介紹

### 🎮 娛樂互動
*   **趣味小遊戲**：內建多款互動指令，提升伺服器成員參與度與活躍度。
*   **自動觸發系統**：根據關鍵詞觸發回應，增添聊天室樂趣。

### 🛡️ 伺服器守護 (防炸保護)
*   **即時防禦**：具備自動踢出、封鎖與禁言功能，有效阻斷炸群攻擊。
*   **管理自動化**：自動清理異常暱稱，確保伺服器整潔。

### 📢 高效管理
*   **公告系統**：透過 `/頂號` 指令快速發送重要通知，確保資訊準確傳達。
*   **狀態監控**：即時分析伺服器運行負載，為管理員提供數據參考。

## 安裝指南

若要將本機器人加入伺服器，請確保已授權以下必要權限：

- **成員管理**：踢出成員、封鎖成員、對成員停權 (Time Out)、管理暱稱。
- **頻道管理**：管理訊息、讀取訊息歷史紀錄、查看頻道。
- **發送訊息**：發送訊息、嵌入連結、附加檔案、提及所有人。
